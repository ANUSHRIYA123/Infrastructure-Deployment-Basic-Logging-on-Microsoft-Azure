# Infrastructure-Deployment-Basic-Logging-on-Microsoft-Azure
This Minor Project focuses on deploying a small-scale enterprise cloud infrastructure using Microsoft Azure. The project simulates a real-world company environment where students act as the Infrastructure Team, responsible for deploying servers, networking, and logging mechanisms.
